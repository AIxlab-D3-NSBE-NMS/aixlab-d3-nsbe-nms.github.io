# Experiment setup and execution

Audience: researchers and operators running studies. Read often, sometimes under time pressure right before a session.

## Setting up a new experiment
Creating an inventory file for the experiment
Selecting which workstations and devices are involved
Preparing experiment files for upload
Dry run on a single workstation before going fleet-wide

## Pre-session preparation
Pre-session checklist (workstations on, devices connected, NAS reachable)
Selecting the correct inventory file
Uploading experiment files to the fleet

## Running a session
Test recording
Start recording
During-task monitoring (what to watch, what to ignore)
Stop recording

## Closing a session
Data backup to the NAS
Verifying files landed correctly
Workstation cleanup
Post-session checklist

### Session-time troubleshooting
   Recording device not detected
   A workstation didn't start/stop with the others
   Backup step partially failed
   What to do if a session has to be aborted mid-task
